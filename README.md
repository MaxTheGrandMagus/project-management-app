<h2>🛠️ Installation Steps:</h2>

<p>1. Clone a Github Repository:</p>

```
git clone https://github.com/MaxTheGrandMagus/project-management-app.git
```

<p>2. Install dependencies:</p>

```
npm install
```

<p>3. Run project with a dev server:</p>

```
npm start
```
